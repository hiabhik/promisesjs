Q4.4.Create a promise function accepting a argument, if yes is passed to the function then it should go to resolved state and print Promise Resolved, and if nothing is passed then it should go to reject state and catch the error and print Promise Rejected 

            let x=new Promise((resolve,reject) =>{
                let a="yes"
                if(a=="yes"){
                    resolve("Promise Resolved")
                }
                else reject("Promise Rejected")
            })
            
            x.then((a)=>{console.log(a)}).catch((a)=>{console.log(a)})
